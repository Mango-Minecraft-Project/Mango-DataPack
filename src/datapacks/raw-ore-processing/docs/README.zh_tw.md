## 說明

此資料包新增了原礦塊的熔爐及高爐配方  
可將原礦塊以9倍慢的方式燒成礦物塊  
經驗也會以9倍給予  

## 支援的模組

- create
- crossroads
- mekanism
- mythicmetals
- simpleores
- tconstruct
- thermal
